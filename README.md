# InternLoop (Frontend)

React + TypeScript + Vite + Tailwind demo app scaffold for InternLoop.

Quick start

1. Install dependencies:

```powershell
cd d:/project-intern; npm install
```

2. Run dev server:

```powershell
npm run dev
```

Open `http://localhost:5173`.

Features

- Mock auth (student/company) with local state
- Student and Company dashboards
- Project posting, applying, and submission flows simulated in local state
- Tailwind-based styling with theme colors for student (teal) and company (orange)

Notes

This project uses only local state and mock data in `src/data` for demo purposes.
